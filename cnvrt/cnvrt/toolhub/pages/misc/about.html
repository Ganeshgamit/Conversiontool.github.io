About ToolHub

ToolHub is a simple, fast, and secure toolkit for everyday file tasks — starting with PDFs and images.

What is ToolHub?

A lightweight, browser-first set of tools for converting, merging, splitting, compressing PDFs, and resizing or converting images. No installs. Works on your localhost now; ready for hosting later.

  • PDF → JPG/PNG
  • Merge / Split / Compress PDF
  • Resize images (px / % / presets)
  • Convert JPG / PNG / WebP

Security & Privacy

This demo keeps processing client-side where possible and simulates progress for actions that need a backend. In production, ensure files are encrypted in transit and auto-deleted after processing.

  • HTTPS everywhere
  • Temporary object storage with TTL (e.g., 1 hour)
  • Strict CORS + signed download URLs

Roadmap (short)

FAQ

Is this free?

Yes, the demo is free. Production can offer a free tier + optional Pro plan.

Does it upload my files?

The current demo simulates processing locally. A real backend should auto-delete files after processing.

Which formats are supported?

PDF, JPG, PNG, WebP now (demo UI). More can be added with a backend.
